# hajioto-hp

## About

「拝啓、はじまりの音へ」公式ホームページのリポジトリ

## Getting Started

1. `.env.local` ファイルを作成し、以下の内容を記述する

   ```env
   SERVICE_DOMAIN=xxx
   API_KEY=xxx
   ```

   xxx の部分は microCMS を参照。

2. `yarn install`
3. `yarn dev`

localhost:3000 でアクセス可能。
